# Non-Simulator (At-School) Wiring:

# DHT11 Sensor
VCC → 5V
DATA → D2
NC → (leave unconnected)
GND → GND

| Component        | Pin Label on Component | Connect to Arduino Pin |
| ---------------- | ---------------------- | ---------------------- |
| **DHT11/22**     | VCC                    | 5V                     |
|                  | DATA                   | D2                     |
|                  | NC                    | unconnected            |
|                  | GND                    | GND                    |
| **OLED Display** | VCC                    | 5V                     |
|                  | GND                    | GND                    |
|                  | SDA                    | A4                     |
|                  | SCL                    | A5                     |
| **Piezo Buzzer** | +                      | Pin 8                  |
|                  | -                      | GND                    |
| **LED**          | Anode (+)              | Pin 9                  |
|                  | Cathode (–)            | GND (via resistor)     |
| **Button**       | One pin                | Pin 7                  |
|                  | Other pin              | GND                    |
