# DHT11 Sensor
VCC → 5V
SDA → D2
NC → (leave unconnected)
GND → GND

# Piezo Buzzer
BZ1:1 → D8
BZ1:2 → GND
Red LED
LED1:A → D9
LED1:C → GND

# Push Button (4 pins)

BTN1:1.L → D7
BTN1:2.L → GND
BTN1:1.R → (no connection)
BTN1:2.R → (no connection)

# OLED Display (SSD1306 I2C)

VCC → 5V
GND → GND
SDA → A4
SCL → A5
